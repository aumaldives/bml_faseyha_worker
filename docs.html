BML API

Base URL: · interactive tester

POST /api/create_session

Exchange a BML OAuth callback URL (containing ?code=...) for tokens. On success the access & refresh tokens auto-fill above and are saved in your browser.

Step 1 · Open BML Login ↗ — log in, then paste the redirected callback URL below.


  


  
  

    
POST /api/refresh

    
Get a fresh access token from a refresh token. Auto-fills the access token above on success.

    
    
    
    

  


  
  

    
GET /api/dashboard

    
Mobile dashboard for the authenticated user. Uses the access token above.

    
    

  


  
  

    
GET /api/history/{id}

    
Today's transaction history for an account id.

    
    
    
    

  


  
  

    
GET /api/statement/{id}/{from}/{to}

    
Account statement between two dates. The bank returns CSV text (not JSON) — rendered as a table here; the raw API response is the CSV itself.

    
    
    

      

      

    

    
    

  


  
  

    
GET /api/notifications

    
Notifications list. Query params (if any) are forwarded upstream.
